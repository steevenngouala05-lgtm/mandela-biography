# Biographie de Nelson Mandela

## Description
Ce projet est un site web présentant la biographie de Nelson Mandela.

## Technologies utilisées
- HTML
- CSS
- JavaScript

## Structure du projet
- index.html : page principale
- styles.css : mise en forme
- script.js : fonctionnalités JavaScript
- photos/ : images du site

## Lancer le projet

1. Télécharger le projet ou le cloner :
   git clone https://github.com/steevenngouala05-lgtm/mandela-biography.git

2. Ouvrir le dossier.

3. Double-cliquer sur index.html
ou utiliser l'extension *Live Server* dans Visual Studio Code.
